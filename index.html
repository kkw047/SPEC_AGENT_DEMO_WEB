Patent Drafting Assistant

발명자료를 출원명세서 초안으로 정리합니다.

SPEC Agent는 발명자의 아이디어와 상담 메모를 명세서 항목으로 구조화하고, 부족한 정보는 질문으로 확인한 뒤 검토용 초안을 생성하는 MVP Agent입니다.

Core Concept

SPEC Agent의 핵심 내용

MVP Agent가 다루는 대상, 출력, 통제 범위를 먼저 정리한다.

업무 대상 출원 준비 자료 정리

발명 아이디어, 구성요소, 효과, 도면 설명을 명세서 초안에 필요한 형태로 정리한다.

핵심 출력 검토용 초안

최종 출원이 아니라 전문가가 확인할 수 있는 초안과 검토 항목을 제공한다.

통제 범위 Human-in-the-loop

특허성 판단, 청구범위 확정, 자동 출원은 수행하지 않고 사람의 검토를 남긴다.

Workflow

업무 흐름도

발명자료가 검토용 출원명세서 초안으로 바뀌는 전체 흐름이다.

1
발명자료 입력

사용자가 아이디어 설명, 기존 방식의 문제점, 구성요소, 작동 방식, 효과, 도면 스케치, 상담 메모를 입력한다.

2
명세서 항목 구조화

Agent가 입력자료를 기술분야, 배경기술, 해결하려는 과제, 해결수단, 효과, 실시예, 도면 설명으로 분류한다.

3
누락 정보 점검

구성요소 연결 관계, 효과 근거, 도면 설명, 실시 가능성처럼 초안 작성에 필요한 정보가 부족한지 확인한다.

4
추가 질문과 보완

부족한 내용은 임의로 채우지 않고 사용자에게 질문으로 제시하며, 사용자의 답변을 다시 반영한다.

5
RAG 참고자료 검색

KIPRIS 공개문헌, 특허로 안내자료, 법령, 지식재산처 공식자료를 참고자료로 연결한다.

6
초안 생성 및 검토 표시

검토용 초안, 추가 질문 목록, 누락 자료, 전문가 검토 필요 항목을 분리하여 출력한다.

Prompt

Prompt 설계

초안 작성 보조 역할과 수행 금지 범위를 명확히 구분한다.

Prompt는 초안 작성 보조 역할과 수행 금지 범위를 명확히 구분한다. Agent는 사용자가 제공한 자료만 발명 내용의 근거로 사용하고, 근거가 부족한 부분은 초안 본문이 아니라 추가 질문 또는 전문가 검토 필요 항목으로 분리한다.

역할: 특허 출원 준비를 돕는 문서 정리 Agent
입력: 발명 아이디어, 구성요소, 작동 방식, 효과, 도면 설명, 상담 메모
출력: 기술분야, 배경기술, 해결 과제, 해결수단, 효과, 실시예, 도면 설명, 추가 질문, 누락 자료, 검토 필요 항목
제한: 사용자 입력에 없는 기술 구성, 실험결과, 정량 효과를 생성하지 않는다.
제한: 특허성 판단, 청구범위 확정, 자동 출원은 수행하지 않는다.
